Nodes and Flow:
External Patients -> Front Door -> Application Gateway.
Hospital Networks -> VPN Gateway -> Firewall.

Clusters and Nesting:
- Transit Hub VNet:
  - Ingress Subnet contains Application Gateway.
  - Hybrid Network Subnet contains VPN Gateway.
  - Egress Security Subnet contains Firewall.

- Clinical API Spoke VNet:
  - Patient Portal Subnet contains App Services.
  - FHIR Gateway Subnet contains API Management.

- Core Processing Spoke VNet:
  - Microservices Subnet contains AKS.
  - Messaging Subnet contains Service Bus.
  - Event Driven Subnet contains Function Apps.

- Secure Data and AI Spoke VNet:
  - Medical Imaging Subnet contains Storage Accounts.
  - Structured Billing Subnet contains SQL Databases.
  - NoSQL Records Subnet contains Cosmos DB.
  - Diagnostics Subnet contains Cognitive Services.

- Compliance and Platform Spoke VNet:
  - Encryption Subnet contains Key Vault.
  - Trusted Image Subnet contains Container Registry.

- Audit and Observability (External):
  - Application Telemetry contains App Insights.
  - Security Audit Logs contains Log Analytics Workspace.
